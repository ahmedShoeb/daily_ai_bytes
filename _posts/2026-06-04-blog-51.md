## Key Points

Mistral released Search Toolkit in public preview - a composable framework for building production search pipelines for AI applications
The toolkit combines ingestion, retrieval, and evaluation into a single framework with shared interfaces
Aims to solve common enterprise search problems where teams spend weeks on infrastructure integration instead of improving search quality
Supports multiple search scenarios including enterprise search across wikis, ticket systems, and document repositories
Provides tools for evaluating RAG systems to isolate retrieval quality from generation quality with built-in metrics
Enables hybrid search with both BM25 sparse retrieval and dense embedding-based retrieval
Integrates with agent workflows, providing indexed search alongside live data retrieval through MCP integrations
Ships with connectors for Vespa indexing and includes sample data pipelines for quick setup
Has been battle-tested across financial services, manufacturing, public sector, and media & entertainment industries
CMA CGM uses it alongside Voxtral for fake news detection with 15-second end-to-end processing time
