## Key Points

- PrismML has announced Bonsai 27B, based on Qwen3.6 27B, which is the first 27B-class multimodal model capable of running on a phone
- The model comes in two variants: Ternary Bonsai 27B (5.9 GB) for quality-oriented use on laptops, and 1-bit Bonsai 27B (3.9 GB) for footprint-oriented use on phones
- Both variants feature multimodal capabilities, a 262K-token context, and support speculative decoding for acceleration
- The ternary variant retains 95% of full-precision baseline performance across 15 benchmarks, while the 1-bit variant retains 90%
- This breakthrough enables agentic workloads to run locally on devices, allowing for zero marginal cost for multi-step loops and keeping user data private
- The model achieves speeds up to 163 tok/s in 1-bit and 134 tok/s in ternary on high-end GPUs
- Bonsai 27B runs natively on Apple devices via MLX and NVIDIA GPUs via CUDA
- The model is available under Apache 2.0 License with a free developer preview API currently offered
- This represents a significant paradigm shift for AI deployment, enabling on-device agents that can work offline with private local data
