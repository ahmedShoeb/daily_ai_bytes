## Key Points

- xAI is launching 'Skills' for Grok, a feature that allows the AI to remember user preferences, formatting rules, and workflow steps across all conversations
- Each Grok account comes with built-in skills from xAI that require no setup, and users can customize or override these with their own versions
- Grok can now generate production-ready Word documents, PowerPoint slides, Excel files, and PDFs with full formatting and styling
- Users can create complete slide decks for pitches, quarterly reviews, and workshops with visual hierarchy and speaker notes
- The system can generate polished invoices, contracts, proposals, and resumes with headings, tables, bullet points, and consistent styling
- Grok can create formatted spreadsheets with formulas, color-coded sections, and structure for budgets, dashboards, and forecasts
- Users can create PDF reports, proposals, and research papers formatted for professional submission
- Creating a new skill takes seconds - users can describe it in conversation, upload a file, or write one from scratch
- Grok can save learned workflows and formats as skills, eliminating the need to re-explain preferences each time
- The Skills feature is now live on Grok 4.3 and available across grok.com, iOS, and Android platforms
