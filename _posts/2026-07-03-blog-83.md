## Key Points

- Meituan has launched LongCat-2.0, a 1.6 trillion-parameter Mixture-of-Experts model with approximately 48 billion active parameters per token
- The model is designed for agentic coding, tool use, long-context work, automated workflows, and complex instruction execution
- Features a 1 million-token context window and maximum output length of 128K tokens via the LongCat API Platform
- Available through OpenAI-compatible and Anthropic-compatible API formats with support for various code workflows
- Pretrained across more than 35 trillion tokens using AI ASIC superpods
- Uses LongCat Sparse Attention for long-horizon tasks and trained on hundreds of billions of tokens of 1M-context data
- Pay-as-you-go pricing: $0.75 per 1M uncached input tokens, $0.015 per 1M cached input tokens, $2.95 per 1M output tokens
- The release is currently API-only with public GitHub repository under MIT license, but model weights are forthcoming
- LongCat-2.0 is revealed as the full model behind Owl Alpha, previously undisclosed model running on OpenRouter
- Part of Meituan's broader AI portfolio including multimodal and agent-focused models for physical-world applications
