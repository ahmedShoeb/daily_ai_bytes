## Key Points

The performance of large language model (LLM) systems depends on both model weights and their harness - the code that determines what information to store, retrieve, and present to the model
Current harnesses are largely designed by hand and existing text optimizers compress feedback too aggressively for this setting
Meta-Harness is an outer-loop system that searches over harness code for LLM applications using an agentic proposer that accesses source code, scores, and execution traces through a filesystem
On online text classification, Meta-Harness improves over state-of-the-art context management by 7.7 points while using 4x fewer context tokens
On retrieval-augmented math reasoning, a single discovered harness improves accuracy on 200 IMO-level problems by 4.7 points on average across five held-out models
On agentic coding, discovered harnesses surpass the best hand-engineered baselines on TerminalBench-2
The research demonstrates that richer access to prior experience enables automated harness engineering
The paper was submitted on March 30, 2026, and is categorized under Computer Science > Artificial Intelligence
Meta-Harness represents a novel approach to optimizing the often-overlooked but critical component of LLM systems
