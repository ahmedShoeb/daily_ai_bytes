## Key Points

- Anthropic is preparing a major overhaul of Claude Code desktop experience called "Epitaxy"
- The update introduces a power-user interface with dedicated panels for Plan, Tasks, and Diffs
- New Coordinator Mode allows Claude to act as an orchestrator delegating work to parallel sub-agents
- Users can preview running code directly in the app and work across multiple repositories simultaneously
- Custom agents can be created within the app, lowering barrier for non-CLI users
- The update draws comparisons to OpenAI's Codex "Magic TODO" task system
- Both companies appear ready to ship desktop updates as early as next week
- Competition has shifted from model benchmarks to developer workflow integration
