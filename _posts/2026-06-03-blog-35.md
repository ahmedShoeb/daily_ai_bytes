## Key Points

- xAI has released grok-build-0.1, the company's fastest coding model, now available in public beta via the xAI API.
- The model is specifically trained for agentic coding tasks including web development, debugging, and MCP (Model Context Protocol) support.
- grok-build-0.1 powers the Grok Build CLI.
- The model operates at blazing-fast speeds of 100+ tokens per second.
- Pricing is set at $1 per million tokens for input and $2 per million tokens for output.
- Beyond coding tasks, the model also serves as a fast and economical option for general-purpose agentic and tool calling use cases.
- The model performs best when used with agentic harnesses like Grok Build, Cursor, Hermes Agent, OpenClaw, Kilo Code, or OpenCode.
- grok-build-0.1 is also accessible via OpenRouter and Vercel AI Gateway platforms.
