## Key Points

- OpenShell is NVIDIA's sandboxed execution environment for autonomous AI agents that protects data, credentials, and infrastructure
- It uses declarative YAML policies to prevent unauthorized file access, data exfiltration, and uncontrolled network activity
- Currently in alpha stage with single-player mode focused on getting individual developer environments up and running
- The system isolates each sandbox in its own container with policy-enforced egress routing and defense across four layers: filesystem, network, process, and inference
- Features GPU passthrough support for local inference and fine-tuning workloads when creating sandboxes with the --gpu flag
- Includes pre-configured support for popular AI agents like Claude Code, OpenCode, Codex, and community images like OpenClaw and Ollama
- Offers agent skills for cluster debugging, policy generation, security review, and other workflow automation tasks
- Includes a real-time terminal UI dashboard for monitoring gateways, sandboxes, and providers inspired by k9s
- Built as an agent-first project where agents are expected collaborators with built-in skills for investigation, diagnosis, and prototyping
- All outbound connections are intercepted by the policy engine which can allow, route for inference, or deny requests based on policy
