## Key Points

xAI has launched standalone Grok Speech to Text (STT) and Grok Text to Speech (TTS) APIs for developers
These APIs are built on the same technology stack that powers Grok Voice, Tesla vehicles, and Starlink customer support
Grok STT offers high accuracy with advanced features including word-level timestamps, speaker diarization, and multichannel support
The speech-to-text API includes intelligent Inverse Text Normalization that correctly handles numbers, dates, currencies, and more
Grok STT outperforms competitors like ElevenLabs, Deepgram, and AssemblyAI in benchmark tests across multiple domains
In benchmark tests, Grok STT achieved 5.0% Word Error Rate for phone call entities vs 12.0-21.3% for competitors
The TTS API includes speech tags that allow developers to add natural prosody and emotion using simple markup
Speech tags include features like [laugh], [sigh], [whisper], <emphasis>, <slow>, and <pause>
Pricing is $0.20 per hour for streaming STT and $4.20 per 1 million characters for TTS
Both APIs support 25+ languages and enable seamless language switching without interruptions
