## Key Points

Canva announced Canva AI 2.0, billed as its most significant update since launching in 2013
The first 1 million users can access the research preview starting today
The new AI assistant features a conversational interface that generates designs from descriptions
Under the hood, an orchestration layer allows the model to use all Canva's tools for complex, multi-step tasks
The system avoids common AI image generation pitfalls by allowing every visual element to be edited individually
Canva AI includes persistent memory that learns users' personal taste and style over time
The AI can now connect to other apps like Notion, Slack, Zoom, Gmail, Google Calendar, and others
Users can schedule tasks for Canva AI to complete in the background
The coding function has been upgraded to support HTML imports for editing AI-generated experiences
Access to Canva's AI features remains free, but a new AI Pass add-on increases rate limits for heavy users
