## Key Points

- Claude Tag is Anthropic's new feature that allows teams to work with Claude AI as a collaborative team member on Slack
- Claude can be tagged in Slack channels and delegated tasks while team members focus on other work
- The system builds context by remembering relevant information from channels it's in and can plan out future tasks
- Currently 65% of Anthropic's product team's code is created by their internal version of Claude Tag
- Claude Tag is available today in beta for Claude Enterprise and Team customers
- Key advantages include: multiplayer functionality (multiple users can interact with the same Claude), learning over time from channel context, proactive/ambient behavior, and asynchronous task completion
- Administrators can tightly control Claude's access to sensitive data and tools, creating separate Claude identities for different teams/uses
- Claude Tag works with Opus 4.8 and replaces the existing Claude in Slack app, with a 30-day migration period for administrators
- The system includes spend limits and logging capabilities for administrators to monitor usage
