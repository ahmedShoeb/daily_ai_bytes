## Key Points

- This is a GitHub repository for an autonomous AI research framework that allows AI agents to experiment with neural network training autonomously overnight
- The core idea is to give an AI agent a small LLM training setup and let it modify code, train for 5 minutes, check results, and repeat automatically
- The framework has three main files: prepare.py (fixed constants and utilities), train.py (the single file that agents modify), and program.md (instructions for AI agents)
- Training runs for a fixed 5-minute wall clock time budget regardless of compute, with validation bits per byte (val_bpb) as the metric
- The setup is designed for single NVIDIA GPU environments but has community forks for MacOS and Windows platforms
- The repository includes guidance for running on smaller compute platforms with recommendations for adjusting hyperparameters like vocab_size, sequence length, and model depth
- The project is created by Andrej Karpathy and represents an exploration of autonomous AI research where agents self-modify code to improve models
- The framework is deliberately minimal with the goal of enabling approximately 12 experiments per hour and 100 experiments overnight
