## Key Points

Lance is a lightweight unified multimodal model developed by ByteDance that supports image and video understanding, generation, and editing within a single framework
The model is remarkably efficient with only 3B active parameters but delivers strong performance across multiple benchmarks
Lance was trained entirely from scratch using a staged multi-task recipe within a budget of 128 A100 GPUs
The model supports six key tasks: text-to-image generation, text-to-video generation, image editing, video editing, image understanding, and video understanding
In benchmark tests, Lance achieves an impressive 85.11 score on VBench for video generation, outperforming larger models like TUNA and Wan2.1-T2V
For image generation benchmarks, Lance scores 84.67 on DPG-Bench and 0.90 on GenEval, competitive with much larger models
The model achieves a 7.30 average score on GEdit-Bench for image editing tasks
Lance is open-source and available on Hugging Face with detailed installation instructions and example configurations for different tasks
The repository includes ready-to-run benchmark scripts and supports inference on GPUs with at least 40GB VRAM
The project aims to demonstrate that efficient, unified multimodal models can perform competitively with larger specialized models
