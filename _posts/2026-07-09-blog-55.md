## Key Points

- As of July 2026, there are 35 actively maintained CLI coding agents in a crowded but rapidly evolving field.
- The terminal emerged as an unlikely winner for serious coding agent usage, with adoption in CI, SSH, and GUI-less machines.
- The competitive landscape includes lab agents from model vendors (Claude Code, Codex CLI, Gemini CLI), platform agents (Copilot CLI, Cursor CLI), and open-source harnesses (OpenCode, Pi, oh-my-pi).
- Claude Code (Anthropic) continues to set industry standards with features like agent teams, hooks, and skills, while Codex CLI (OpenAI) offers open-source, OS-sandboxed alternatives.
- Google transitioned Gemini CLI users to the closed-source Antigravity CLI in June 2026, highlighting the instability of free tier offerings.
- Key trends include open-weight models (GLM-5.2, DeepSeek V4, Kimi K2.7 Code) closing the performance gap with frontier models while costing significantly less.
- Feature leaders include: Omp for editing precision and orchestration, Claude Code for teams coordination, DeepSeek-Reasonix for cost engineering, and OpenCode for provider breadth.
- Git workflows have matured with tools like Omp offering advanced commit splitting and conflict resolution, and Aider pioneering atomic commits.
- Safety considerations vary widely: Codex CLI offers both open-source and OS-sandboxed security, while closed agents like Claude Code and Copilot CLI prioritize different trust models.
- The author recommends using Claude Code, Codex CLI, and Omp as peer tools for serious work, with OpenCode as the portable base for BYOK usage, emphasizing that clear task definitions and repository hygiene matter more than tool choice.
