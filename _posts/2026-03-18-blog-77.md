## Key Points

Introduces Attention Residuals (AttnRes), replacing traditional depth-wise recurrence with learned, input-dependent attention over preceding layers
Networks can now selectively retrieve past representations, mitigating dilution and hidden-state growth issues
Introduces Block AttnRes that partitions layers into compressed blocks for practical cross-layer attention at scale
Achieves 1.25x compute advantage with less than 2% inference latency overhead as an efficient drop-in replacement
Validated on Kimi Linear architecture (48B total, 3B activated parameters) with consistent downstream performance gains
Scaling law experiments show consistent 1.25× compute advantage across varying model sizes
Analysis reveals AttnRes mitigates hidden-state magnitude growth and yields more uniform gradient distribution
Introduces Kimi K2: 1T total / 32B active MoE open-source agentic model
Kimi K2 achieves SOTA performance on SWE Bench Verified, Tau2 & AceBench among open models
Strong coding and agentic task capabilities with multimodal and thought-mode not currently supported
API pricing: $0.15/million input tokens (cache hit), $0.60/million input tokens (cache miss), $2.50/million output tokens
Plans to open-source base pretrained model and reinforcement-learned model underlying Kimi-Researcher in coming months
