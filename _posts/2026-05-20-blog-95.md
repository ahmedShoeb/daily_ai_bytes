## Key Points

Introduces Lighthouse Attention, a selection-based hierarchical attention mechanism that dramatically speeds up long-context pretraining
Runs ~17x faster than standard attention at 512K context on a single B200 GPU while maintaining model quality
Uses symmetric pooling of Q, K, and V across a multi-resolution pyramid with parameter-free scoring based on per-head L2 norms
Achieves 1.4-1.7x end-to-end pretraining speedup at 98K context compared to dense baseline
Employs a two-stage training approach: Lighthouse selection for most training, then brief SDPA resume to ensure inference-time dense attention capability
Successfully trained 530M parameter Llama-3 models with 1M-token context across 32 Blackwell GPUs using context parallelism
Every recovered Lighthouse run matches or beats the dense-from-scratch baseline at the same token budget
Code is available on GitHub and implementation requires only about 600 lines of modifications on top of existing frameworks
