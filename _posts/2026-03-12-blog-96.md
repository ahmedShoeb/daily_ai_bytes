## Key Points

Penguin-VL is a compact vision-language model family from Tencent AI Lab designed to push multimodal efficiency limits by redesigning the vision encoder rather than scaling data or model size
Traditional VLMs use CLIP or SigLIP contrastive learning which favors coarse category-level invariances, while Penguin-VL argues this is suboptimal for fine-grained tasks like OCR, document understanding, and dense captioning
Penguin-VL introduces Penguin-Encoder, a vision encoder initialized from a text-only LLM (like Qwen3-0.6B), bringing the visual backbone closer to language model representation space for more data-efficient learning
The framework includes converting causal attention to bidirectional attention, adding 2D-RoPE for variable-resolution vision tokens, mixed-supervision pretraining, and temporal redundancy-aware token compression for video efficiency
Penguin-VL-2B delivers strong accuracy-efficiency tradeoffs, with especially solid gains on OCR-heavy and reasoning-heavy tasks requiring fine-grained visual understanding
The project released Penguin-VL-2B, Penguin-VL-8B, and Penguin Vision Encoder on Hugging Face in March 2026 along with inference code, vLLM plugin, and Gradio demo
The training approach uses a low-to-high resolution curriculum + instruction tuning strategy to balance image and video capabilities at compact scale
Video handling includes Temporal Redundancy-Aware (TRA) token compression that dynamically allocates token budgets across key frames vs intermediate frames under a global token budget for long videos
Available inference options include Transformers inference, vLLM integration, Gradio web UI demo with image/video upload capabilities, and comprehensive Jupyter notebooks
The project is open source under Apache 2.0 License and provides extensive documentation including setup instructions, multiple inference methods, and citation information
