## Key Points

NVIDIA developed Nemotron OCR v2, a multilingual OCR model that's both accurate and fast, achieving 34.7 pages/second on a single A100 GPU.
The solution uses synthetic data generation with 12 million training images across 6 languages to overcome the cost and limitations of manual annotation.
Key improvement: Reduced Normalized Edit Distance (NED) scores from 0.56-0.92 (unusable) down to 0.035-0.069 across non-English languages like Japanese, Korean, Russian, and Chinese.
The model handles 5 languages simultaneously (EN, ZH, JA, KO, RU) with a single unified architecture - no need for language detection or switching models.
The synthetic data pipeline generates pixel-perfect annotations at word, line, and paragraph levels plus relation graphs for reading order understanding.
Architecture uses FOTS design with shared backbone (RegNetX-8GF) enabling feature reuse between detection, recognition, and relational components for speed.
Training data includes mOSCAR multilingual web corpus with realistic text distributions and diverse font pools from Google Fonts and Noto family.
Real-world benchmark shows Nemotron OCR v2 multilingual delivers competitive accuracy at over 28x faster speed than PaddleOCR v5.
Dataset (12M+ images) and model are publicly available under NVIDIA Open Model License with demo available in browser Space.
Pipeline is extensible - adding new languages only requires source text and fonts, making it scalable to cover many more languages.
