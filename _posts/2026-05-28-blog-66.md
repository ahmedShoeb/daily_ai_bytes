## Key Points

X.ai has launched Grok Build in early beta, a powerful new coding agent and CLI tool designed for professional software engineering tasks
The tool is available exclusively to SuperGrok and X Premium Plus subscribers as part of their subscription
Grok Build can be installed with a single command: curl -fsSL https://x.ai/cli/install.sh | bash
For complex tasks, it offers a 'plan mode' where users can preview, approve, comment on, or rewrite steps before execution begins
All changes made by the agent show up as clean diffs for easy review and version control
The tool automatically integrates with existing development workflows including AGENTS.md, plugins, hooks, skills, and MCP servers
For larger projects, Grok Build can delegate work to specialized subagents that run in parallel and supports deep worktree integrations
It includes a headless mode (-p) for running agents within scripts and automations
The CLI provides full ACP (Agent Communication Protocol) support for building custom bots and agent orchestration applications
X.ai is actively seeking user feedback during this early beta phase and encourages users to submit feedback via /feedback command in the CLI
