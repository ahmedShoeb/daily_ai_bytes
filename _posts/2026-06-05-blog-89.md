## Key Points

- Wall Attention is a novel attention variant with per-channel, per-timestep multiplicative decay built into the QK inner product
- The repository provides two optimized kernels: wall_attn for training/prefill with fused forward+backward operations, and wall_attn_decode for efficient single-step decoding
- The system uses a pre-rescaled KV cache that enables cheap autoregressive generation with one small GEMV-like pass per token instead of recomputing prefixes
- Features include GQA (Group Query Attention) support, per-channel decay with exact analytic gradients, attention sink capability, sliding window, and varlen packing
- The decode kernel builds pre-rescaled cache during prefill using per-chunk anchors to keep numerical operations bounded for long context handling
- Implementation uses Triton kernels built on flash-linear-attention infrastructure with BF16/FP32 support and autotuned block sizes for Hopper/Ampere GPUs
