## Key Points

- xAI is developing a credits-based pricing model for its upcoming Grok Build coding environment
- The credits system was discovered hidden within recent builds of the platform, allowing users to monitor balances for coding tasks
- Grok Build will be a two-pronged offering with both local CLI and web interface options
- A unique 'Model Arena' feature allows multiple AI agents to tackle the same task in parallel for output comparison
- The credits system follows industry standards used by OpenAI and Anthropic - subscribers get monthly credits with on-demand purchases available
- The commercial infrastructure is still early in development but shows xAI is preparing for a full rollout
- Grok Build aims to compete with OpenAI's Codex and Anthropic's Claude Code in the AI coding assistant space
