## Key Points

- Android 17 has been released and represents a major shift from an operating system to an 'intelligence system' with AI integration
- Introduces AppFunctions platform API for apps to contribute capabilities as orchestratable tools for AI agents like Google Gemini
- Mandates adaptive-first development standard requiring apps to support all form factors (phones, tablets, foldables, etc.)
- Android development is now Compose-first - all new APIs and tools will be built exclusively for Jetpack Compose
- Implements strict app memory limits based on device RAM to improve performance and reduce background app termination
- Introduces privacy features like temporary session-based access to contacts, local network permissions, and expanded SMS OTP protection
- Adds new media capabilities including VVC video codec support and better hearing aid device support
- Will no longer restart activities by default for typical configuration changes, improving UI smoothness
- Introduces Continue On feature for seamless task transitions between Android devices
- Launches AI-powered developer tools including Jetpack Compose adaptive skill and XML to Compose migration skill
