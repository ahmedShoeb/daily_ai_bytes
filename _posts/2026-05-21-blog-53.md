## Key Points

HRM-Text is a 1B parameter text generation model based on hierarchical recurrent architecture (HRM) with task completion and latent space reasoning
The framework enables foundation model pretraining with ~$1000, requiring 130-600x less compute and 150-900x less data compared to traditional approaches
Two model sizes available: L (0.6B parameters) for ~$800 using 8 H100s for 50 hours, and XL (1B parameters) for ~$1472 using 16 H100s for 46 hours
Achieves strong benchmark performance - the XL model scores 84.7% on GSM8k, 56.5% on MATH, 60.7% on MMLU, and 81.9% on ARC-C
Includes a full pretraining framework with PrefixLM sequence packing, FlashAttention 3 kernels, PyTorch FSDP2 training, and evaluation tooling
Uses hierarchical recurrent architecture that differs from standard Transformers, improving efficiency through innovative attention patterns
Supports multi-node distributed training and provides comprehensive documentation for setup, data preparation, and training workflows
Includes conversion tools to export models to Hugging Face format and supports fine-tuning for instruction following tasks
Open-source project under Apache License 2.0 with active contribution guidelines and community engagement
Based on research paper "HRM-Text: Efficient Pretraining Beyond Scaling" with arXiv preprint available
