## Key Points

- The paper addresses the fragmentation in object counting across domain-specific datasets and task formulations
- Introduces a text-guided object counting formulation where models take an image and natural-language query to return instance-grounded target points
- Created CLOC (Cross-domain Large-scale Object Counting dataset) covering six visual domains with ~220K images, 619 categories, and 15M object instances
- Proposes 'Count Anything' - a generalist model for text-guided object counting using discrete instance points instead of density maps
- Uses dual-granularity instance enumeration: Region-level Sparse Counter for large/sparse targets and Pixel-level Dense Counter for small/crowded targets
- Implements point-centric supervision to learn from heterogeneous annotations
- Shows strong accuracy and multi-domain generalization, outperforming existing open-world counting methods
- Unifies category-conditioned counting with interpretable spatial localization
