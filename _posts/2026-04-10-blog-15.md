## Key Points

- Anthropic has announced Claude Mythos Preview, a new general-purpose language model with strikingly advanced computer security capabilities
- The model performs strongly across various security tasks and can autonomously identify and exploit zero-day vulnerabilities in every major operating system and web browser
- Mythos Preview demonstrated the ability to write complex exploits like JIT heap sprays, sandbox escapes, and sophisticated ROP chains without human intervention
- The model found a 27-year-old bug in OpenBSD and exploited a 17-year-old vulnerability in FreeBSD's NFS server granting root access
- Mythos Preview can chain multiple vulnerabilities together, demonstrating capabilities that previously required expert penetration testers weeks to develop
- The company launched Project Glasswing to use Mythos Preview to secure critical software and prepare the industry for new cybersecurity practices
- Despite not being trained specifically for security tasks, these capabilities emerged from general improvements in code, reasoning, and autonomy
- Anthropic warns that the transitional period will be tumultuous as AI capabilities disrupt the current security equilibrium
- The company provides cryptographic commitments for vulnerabilities and exploits they've discovered, showing the scope of their findings
- Recommendations include using current frontier models for defensive work, shortening patch cycles, and automating incident response pipelines
