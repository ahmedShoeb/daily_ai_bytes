## Key Points

- NVIDIA Nemotron 3 Nano Omni is a new omni-modal understanding model designed for document analysis, multiple image reasoning, automatic speech recognition, long audio-video understanding, agentic computer use, and general reasoning.
- The model achieves best-in-class accuracy on complex document intelligence leaderboards like MMlongbench-Doc and OCRBenchV2, and leads in video and audio leaderboards including WorldSense and DailyOmni.
- It combines a Nemotron 3 hybrid Mamba-Transformer Mixture-of-Experts backbone with C-RADIOv4-H vision encoder and Parakeet-TDT-0.6B-v2 audio encoder for native audio understanding.
- The architecture is designed to handle very long multimodal contexts for dense images, documents, videos, and mixed-modality reasoning, with up to 9x higher throughput than alternatives.
- Nemotron 3 Nano Omni is built for five main workloads: real-world document analysis (100+ page documents), automatic speech recognition, long audio-video understanding, agentic computer use (GUI interactions), and general multimodal reasoning.
- The model uses dynamic resolution processing for dense documents and charts, Conv3D temporal compression for video, and efficient video sampling to reduce latency while maintaining accuracy.
- It includes native audio input processing that can handle up to 5+ hours of audio, allowing audio, visual, and text tokens to be jointly modeled within shared multimodal sequences.
- The model was trained using staged multimodal alignment, context extension, preference optimization, and multimodal reinforcement learning across diverse environments.
- Compared to alternatives, Nemotron 3 Nano Omni delivers 7.4x higher system efficiency for multi-document use cases and 9.2x higher efficiency for video use cases at fixed interactivity thresholds.
- The model checkpoints (BF16, FP8, and NVFP4) are available on Hugging Face, along with technical documentation and training code on GitHub.
