## Key Points

- Anthropic is preparing a major overhaul of Claude's memory system with a dual-mode architecture
- The current "classic" system uses a single summarized note for user information
- The new "Memory Files" system will use multiple structured documents organized by topic, project or context
- This file-based approach allows for larger and more durable user records without overwhelming Claude's context window
- The system would function as a built-in personal wiki that Claude can selectively consult
- The update may be preparation for the upcoming Claude Conway agent release
- A related "Dreams" feature is in beta, which processes memory files asynchronously like REM sleep consolidation
- Dreams merges duplicates, updates stale entries, resolves contradictions and identifies patterns missed during live sessions
- No firm timeline has been given for the memory system overhaul
- Anthropic is also working on smaller UI tweaks in parallel to this major update
