## Key Points

Google has launched new AI capabilities for Android developers: hybrid inference API and support for new Gemini models
Hybrid inference is a new Firebase API that allows apps to dynamically switch between on-device Gemini Nano models and cloud-hosted Gemini models
The hybrid inference API uses a rule-based routing approach with modes like PREFER_ON_DEVICE (falls back to cloud if on-device unavailable) and PREFER_IN_CLOUD (falls back to on-device if offline)
Two new Gemini models are now available: Nano Banana Pro for professional image generation and Gemini 3.1 Flash-Lite for advanced use cases
Nano Banana Pro (Gemini 3 Pro Image) can render high-fidelity text, simulate different handwriting styles, and generate precise, detailed images
The hybrid inference API is currently experimental and supports single-turn text generation based on text or single Bitmap image inputs
Google has updated its Magic Selfie sample to use Nano Banana 2, which now handles background segmentation directly through image generation
Gemini 3.1 Flash-Lite is currently in preview and offers latency comparable to Gemini 2.5 Flash-Lite with more advanced capabilities
Gemini Flash-Lite models have been popular with Android developers for their good quality/latency ratio and low inference cost
Android documentation and sample code are available for developers to explore these new AI capabilities
