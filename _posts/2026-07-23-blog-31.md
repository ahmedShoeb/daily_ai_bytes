## Key Points

- Anthropic's Conway experiment is ending on July 24th, 2025 at 5 pm PT, with testers asked to export their data
- Conway is an always-on agent that first surfaced this spring as a standalone instance capable of running Claude Code, operating a browser, and sending notifications
- There are two possible interpretations for the shutdown: either Conway doesn't fit Anthropic's roadmap, or internal testing is wrapping up before a broader release
- The first possibility suggests Anthropic may focus on desktop agents like Cowork instead of remote containers
- The second possibility suggests Conway could become available to Max subscribers first, fitting Anthropic's typical rollout pattern
- Conway could become a remote container across Anthropic's apps where users install plugins, hand off work, and configure webhooks
- A related 'UI tabs' feature is expected to let users create custom control panels for Conway
- The outcome will reveal whether Anthropic believes persistent AI agents should be on user devices or in the cloud
- This decision is significant as rival AI labs are also developing their own always-on agent projects
