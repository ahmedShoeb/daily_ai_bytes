## Key Points

- Progress Agentic RAG offers flexible pricing with three main tiers: Starter ($700/month), Pro ($1,925/month), and Enterprise (custom pricing)
- All plans include unlimited named users and data augmentation agents
- Starter plan includes 5GB or 15K resources, max file size of 750MB (text files only)
- Pro plan includes 25GB or 80K resources, max file size of 1.5GB (all file types)
- Enterprise plan offers unlimited scalability with unlimited indexed data and file sizes
- All plans include consumption price of $0.008 per token (first 10,000 tokens/month free)
- LLM availability varies: Starter includes ChatGPT 4o and ChatGPT 4o mini; Pro and Enterprise include all LLMs
- Knowledge boxes included: 1 for Starter, 2 for Pro (up to 8 available), 12 for Enterprise
- 14-day free trial available for exploring the platform
