## Key Points

OpenAI is building a new platform for persistent, autonomous agents within ChatGPT under internal codename Hermes
The new section will allow users to create agents that operate continuously as teammates rather than one-off task runners
Users can assemble custom workflows, attach skills, plug in connectors, and wire agents into messaging surfaces outside ChatGPT
Features include task scheduling so agents can act on cadence, events, and incoming messages rather than waiting for prompts
Placeholder examples reference roles like CTO or CPO, suggesting users could orchestrate multiple agents to function like a small AI-run organization
This positions OpenAI to compete with Notion, which has been the most visible player in custom agents with its Custom Agents feature
The platform would bring agent capabilities directly to ChatGPT's hundreds of millions of existing users who already use skills, connectors, and tasks
Hermes branding appears consistently throughout the web app, indicating active development rather than an experiment
No confirmed release window yet, but the platform represents a shift from single assistant to multiple always-on agents working in parallel
