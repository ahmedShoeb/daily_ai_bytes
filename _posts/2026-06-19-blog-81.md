## Key Points

- Anthropic has paused major pricing changes for its Claude Agent SDK that were scheduled to take effect on June 15
- The proposed changes would have treated Agent SDK usage separately from standard Claude usage and billed it at API rates
- Under the new plan, SDK users would have received only a monthly usage credit equal to their subscription price
- The current system allows SDK usage to count against generous weekly caps in existing Claude subscriptions
- Power users currently get much more value from subscriptions than they would from equivalent API usage
- One analysis shows Claude Opus subscribers start saving money after just 2-3 messages per day
- The pricing change would have substantially increased costs for heavy users and third-party app developers
