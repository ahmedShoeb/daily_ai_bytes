## Key Points

- Mistral releases Leanstral, the first open-source code agent designed for Lean 4, a proof assistant capable of expressing complex mathematical objects and software specifications.
- Leanstral is highly efficient with 6B active parameters using a sparse architecture, optimized for proof engineering tasks.
- The model is released under Apache 2.0 license with multiple access methods: through Mistral Vibe, free API endpoint, and downloadable weights.
- Leanstral significantly outperforms larger open-source competitors like GLM5-744B and Kimi-K2.5-1T, achieving better scores with far fewer parameters and computational costs.
- Compared to Claude family models, Leanstral offers competitive performance at a fraction of the price - beating Sonnet by 2.6 points while costing only $36 vs Sonnet's $549.
- The model demonstrates practical utility through case studies: fixing breaking changes in Lean releases and converting/verifying programming language semantics.
- Leanstral successfully diagnosed and fixed a real-world StackExchange issue involving Lean 4.29.0-rc6 breaking changes by changing 'def' to 'abbrev'.
- The model can translate programming language definitions (like from Rocq to Lean) and prove properties about programs.
- Available today with zero-setup in Mistral Vibe via command '/leanstall' or API endpoint 'labs-leanstral-2603'.
- Development includes FLTEval evaluation suite to move beyond competition math benchmarks toward realistic proof engineering scenarios.
