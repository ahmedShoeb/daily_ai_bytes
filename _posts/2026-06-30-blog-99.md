## Key Points

- Researchers have created ALPBench, a comprehensive benchmark for evaluating active learning pipelines on tabular data
- Active learning aims to select the most informative data points to label when budgets for labeled data are limited
- The community has lacked standardized benchmarks for comparing different active learning query strategies, especially in combination with learning algorithms
- ALPBench includes 86 real-world tabular classification datasets and 5 active learning settings, creating 430 active learning problems
- The benchmark facilitates specification, execution, and performance monitoring of active learning pipelines
- Built-in reproducibility measures save exact dataset splits and hyperparameter settings
- To demonstrate utility, researchers evaluated 9 query strategies paired with 8 learning algorithms in 2 different settings
- The work addresses a significant gap in the active learning literature by providing standardized evaluation protocols
