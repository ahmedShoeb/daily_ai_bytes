## Key Points

- Together.ai has released Aurora, an open-source RL-based framework that enables continuous learning directly from live inference traces without interrupting model serving
- Aurora solves the problem of draft model staleness in speculative decoding by creating a serve-to-train flywheel that learns asynchronously
- The system achieves a 1.25x speedup over well-trained static speculators on models like Qwen3 and Llama3
- Aurora demonstrates that online training from scratch can outperform carefully pretrained static baselines
- The framework eliminates costly activation-collection pipelines that can reach petabyte-scale in production environments
- Aurora uses reinforcement learning to maximize acceptance length, which directly maps to decoding speedup
- The system employs a Tree Attention mechanism to efficiently process both accepted and rejected speculative decoding branches
- Aurora successfully adapts to distribution shifts across five different domains including math reasoning, code generation, and finance
- The approach treats speculative decoding as a joint learning-and-serving problem rather than just a modeling challenge
- This represents a fundamental shift from static offline training to dynamic online learning processes for speculative decoding
