## Key Points

- Cohere launched North Mini Code, its first agentic coding model and inaugural member of their next generation of powerful models
- The model is a mixture-of-experts (MoE) with 30B total parameters but only 3B active parameters, making it efficient and hardware-friendly
- North Mini Code is open-source under Apache 2.0 license and available on Hugging Face, Cohere API, Model Vault, and OpenRouter
- The model supports 256K total context with 64K maximum generation length, optimized for code generation and agentic software engineering
- According to Cohere's testing, North Mini Code achieved up to 2.8x higher output throughput than Devstral Small 2 under identical conditions
- The model demonstrated a 30% advantage in inter-token latency and competitive performance across coding benchmarks
- North Mini Code is designed for agentic workflows including understanding/orchestrating sub-agents, mapping system architecture, and running code reviews
- This marks Cohere's first open-source model for developers as part of their mission to make sovereign AI a practical reality
- Cohere plans to continue expanding their ecosystem with more open and sovereign developer models
- The company encourages community feedback to directly shape their future roadmap for sovereign AI development
