## Key Points

- Claude Code is an agentic coding tool that can run shell commands, edit files, and call external services on behalf of users
- The study analyzes Claude Code's architecture by examining its publicly available TypeScript source code
- Researchers identify five core human values motivating the architecture: human decision authority, safety and security, reliable execution, capability amplification, and contextual adaptability
- The system's core is a simple while-loop that calls the model, runs tools, and repeats, but most complexity is in surrounding systems
- Architecture includes a permission system with seven modes and ML-based classifier, five-layer compaction pipeline for context management
- Four extensibility mechanisms: MCP, plugins, skills, and hooks
- Comparison with OpenClaw shows same design questions produce different architectural answers in different deployment contexts
- The paper identifies six open design directions for future agent systems based on empirical, architectural, and policy literature
- The research uses Claude Code as a case study to understand broader design patterns in AI agent systems
- The study contributes to understanding how deployment context shapes AI agent architecture decisions
