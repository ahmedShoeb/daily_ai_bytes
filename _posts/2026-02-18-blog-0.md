## Key Points

OpenRouter offers a free models router that normalizes requests and responses across different AI model providers
The platform supports reasoning-enabled models that can show step-by-step thinking processes
Developers can enable reasoning using the reasoning parameter in API requests
The reasoning_details array in responses shows the model's internal reasoning before the final answer
When continuing conversations, developers should preserve the complete reasoning_details array to maintain context
OpenRouter provides sample code and API documentation for their free models router
The service includes OpenRouter-specific headers that allow apps to appear on OpenRouter leaderboards
Third-party SDKs and frameworks are supported through OpenRouter's frameworks documentation
The platform handles request normalization and response standardization across multiple AI providers
