## Key Points

OpenAI is expanding its Daybreak cybersecurity platform beyond vulnerability discovery into patch automation
The launch includes an updated Codex Security plugin integrated into Codex workflows for comprehensive code scanning
GPT-5.5-Cyber model is being released in limited access specifically for verified cybersecurity defenders
OpenAI introduced Patch the Planet - an open-source security initiative with major projects like cURL, Python, and Go
A new Daybreak Cyber Partner Program aims to embed GPT-5.5-Cyber into existing security products rather than direct access
Since March research preview, Codex Security scanned over 30 million commits across 30,000+ codebases
GPT-5.5-Cyber shows improved performance: 85.6% on CyberGym vs 81.8% for standard GPT-5.5, and 39.5% on ExploitGym vs 25.95%
More than 30 open-source projects have committed to Patch the Planet including major tools like cURL, Python, and freenginx
OpenAI is positioning Daybreak as a complete defensive cyber stack with frontier models, workflows, and security integrations
The release marks a shift from AI cybersecurity as just evaluation metrics to operational pipelines for scanning, fixing, and reviewing vulnerabilities
