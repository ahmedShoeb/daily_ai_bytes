## Key Points

- Anthropic is shipping a new feature called 'Dispatch' in Claude Cowork as a research preview
- Dispatch creates one persistent conversation with Claude that runs on your computer
- Users can message Claude from their phone and come back to finished work
- Because it's Cowork, Claude runs code in a sandbox on the user's machine with files staying local
- Users approve what Claude touches before it acts, maintaining security and control
- Claude gets its own sandboxed Linux VM separate from the user's machine
- On Windows, Hyper-V virtualization technology powers the VM with SYSTEM privileges managed by a separate Go service
- The connection between Claude and VM is secured with Authenticode certificate pinning
- Rolling out now to Max subscribers, with Pro coming in the next few days
- Users need their desktop running for the feature to work
