## Key Points

- FlashDrive is an algorithm-system co-design framework that accelerates Vision-Language-Action (VLA) models for autonomous driving by 4.5×, reducing latency from 716ms to 159ms.
- Traditional autonomous driving systems separate perception and planning, making them brittle in rare scenarios, while VLA models integrate chain-of-thought reasoning into end-to-end driving.
- NVIDIA's Alpamayo 1.5 (10B parameters) takes 716ms per step, far short of real-time driving requirements at roughly 1.4 Hz.
- The framework attacks all four inference stages: vision encoding, prompt prefilling, reasoning token decoding, and action generation via flow matching.
- Key techniques include streaming inference with KV cache reuse (75% vision computation reduction), speculative reasoning with DFlash block diffusion, adaptive-step flow matching, and advanced quantization methods.
- Streaming inference processes only new frames, reusing 75% of computed frames from previous time steps while maintaining accuracy through targeted expert fine-tuning.
- The team uses ParoQuant's scaled pairwise rotation for W4A8 quantization to handle compounding errors in reasoning trace generation.
- System optimizations include CUDA Graphs compilation and kernel fusion to eliminate CPU dispatch overhead and memory round-trips.
- Results show 4.0×-5.7× speedups across NVIDIA platforms including Jetson Thor, RTX 3090, RTX 4090, RTX 5090, and RTX PRO 6000 GPUs.
- Sub-200ms inference on a single GPU makes reasoning-capable VLA models viable for real-time autonomous driving deployment without sacrificing chain-of-thought capabilities.
