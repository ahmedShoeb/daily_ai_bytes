## Key Points

OpenAI is developing a new Scratchpad feature for Codex that allows users to trigger multiple Codex tasks in parallel from a dedicated UI
The Scratchpad will be a key component of the upcoming "Codex Superapp" which enables triggering a broader range of tasks to achieve specific goals
OpenAI is consolidating its product lineup into a single unified application built on top of Codex, potentially absorbing ChatGPT and Atlas browser into one platform
Code references reveal a heartbeat system for maintaining persistent connections with long-running tasks, similar to infrastructure used by OpenClaw
The heartbeat system suggests OpenAI is building support for managed agents - autonomous processes that can run in the background and execute multi-step workflows
This represents a competitive response to Anthropic, which is developing its own managed-agent system codenamed "Conway" for Claude
A single unified app could drastically reduce friction for developers and power users by handling chat, code generation, and web research through persistent agents
OpenAI employees have been posting snowflake emojis, fueling speculation about a model release codenamed "Glacier" (believed to be GPT-5.5)
OpenAI may release both the super app and a new model simultaneously, following their pattern of maximizing impact with coordinated releases
Rollout could happen as soon as the coming days, given OpenAI's tendency toward rapid, sometimes surprise releases
