## Key Points

- Microsoft is planning to move GitHub Copilot from flat-rate subscription models to token-based billing
- This change would affect both the "Copilot for Business" tier and the "Copilot for GitHub Enterprise" plan
- The new system would use a "buckets" model where organizations pay for token usage rather than per-user flat fees
- Microsoft would also reduce rate limits for both tiers, which could impact development workflows
- These changes could make costs less predictable for organizations using Copilot extensively
- The shift appears designed to better align pricing with actual usage patterns
- Some developers may need to adjust their workflows to manage token usage and costs
- The changes reflect Microsoft's evolving strategy for AI-powered coding assistance tools
