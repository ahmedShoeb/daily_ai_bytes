## Key Points

Mellum 2 is a new open-weight 12B-parameter Mixture-of-Experts (MoE) language model specialized for software engineering tasks.
The model has 2.5B active parameters per token, making it efficient to run on commodity GPUs despite its total size.
Architecture features include Grouped-Query Attention with 4 KV heads, Sliding Window Attention, and Multi-Token Prediction head for speculative decoding.
Pre-training used 10.6 trillion tokens through a three-stage curriculum that shifted from web data to curated code and math content.
Post-training included supervised fine-tuning and RLVR reinforcement learning, producing both direct-answer and thinking-trace variants.
The model competes with open-weight baselines in the 4B-14B parameter range while maintaining compute efficiency of a 2.5B dense model.
All checkpoints (base, instruct, and thinking) are released under Apache 2.0 license, including architectural details and training recipes.
