## Key Points

The extension is a Chrome Manifest V3 extension built with React that opens as a side panel alongside active tabs and acts as an AI agent interacting with web pages
Operates in two distinct modes: standard tool-use loop for complex actions and a fast command-based mode with compact single-letter commands for low-latency operations
Uses Chrome DevTools Protocol (CDP v1.3) for all automation including mouse clicks, typing, scrolling, and screenshots
Features a sophisticated permission system with multiple modes (ask, follow-a-plan, skip-all-checks) and different permission types for various actions
Includes comprehensive web interaction tools including navigate, read_page for accessibility tree generation, form_input, find, get_page_text, and javascript execution
Implements tab management through Chrome Tab Groups with visual indicators (loading animations, checkmarks) and automated tab context injection
Supports connections to external MCP servers including Claude Desktop and Claude Code via native messaging protocols
Includes session recording capabilities with rrweb for creating reusable shortcuts that can be scheduled for recurring execution
Manages message size by extracting and replacing older base64 images when sessions exceed approximately 25MB
Employs a nested LLM architecture for natural language element finding and sophisticated DOM interaction capabilities
