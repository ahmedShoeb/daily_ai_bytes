## Key Points

The article presents an experiment comparing standard BM25 search vs. LLM-powered agentic search across two datasets (WANDS and ESCI)
Using GPT-5 with a simple search tool improved BM25 baseline scores from 0.56 to 0.64 on WANDS dataset and from 0.30 to 0.39 on ESCI dataset
The author argues for building simple, transparent search tools rather than complex systems, allowing agents to iteratively learn how to use them effectively
The agentic loop involves prompting an LLM with search specifications, allowing it to call tools, and returning structured outputs with search plans and intent explanations
The experiment supports the hypothesis that reasoning agents work best with simple search tools they can learn to use through iteration
