## Key Points

Google is launching 'Skills in Chrome' - a feature that allows users to save and reuse AI prompts as one-click tools
Users can save frequently used AI prompts from chat history and run them with a single click on any webpage
Skills can be triggered by typing forward slash (/) or clicking the plus sign (+) button in Gemini in Chrome
The feature works across multiple tabs simultaneously - a Skill can run on the current page plus any other selected tabs
Google is providing a library of ready-to-use Skills for common workflows and tasks
Examples of Skills include: calculating protein macros for recipes, generating product spec comparisons, and scanning documents for key information
Users can create custom Skills and edit existing ones to better fit their needs
Skills inherit Chrome's security and privacy protections with confirmation prompts for sensitive actions
Skills are rolling out to Gemini in Chrome on desktop starting today
Saved Skills sync across signed-in Chrome desktop devices and can be managed through the Gemini interface
