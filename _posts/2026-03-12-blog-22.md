## Key Points

- A recent Forbes article claimed Anthropic's $200/month Claude Code Max plan can consume $5,000 in compute costs per user
- The author argues this figure confuses retail API prices with actual compute costs
- API pricing for Opus 4.6 is $5/$25 per million tokens, but actual inference costs are roughly 10% of that
- OpenRouter shows comparable models serving at ~10% of Anthropic's API prices while running profitable businesses
- Heavy users likely cost Anthropic about $500 in actual compute, not $5,000, resulting in a $300 loss, not $4,800
- Most users (95%) don't reach usage caps and average around $180/month in API-equivalent spend
- The $5,000 figure likely applies to Cursor, which pays retail API prices, not Anthropic's internal costs
- Anthropic's unprofitability comes from training costs and research salaries, not inference
- The 'AI inference is a money pit' narrative benefits frontier labs by discouraging competition
- Actual inference economics are revealed by competitive open-weight model providers on platforms like OpenRouter
