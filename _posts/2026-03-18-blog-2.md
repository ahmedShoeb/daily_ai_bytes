## Key Points

Cerebras is deploying its CS-3 systems in AWS data centers, making the world's fastest AI inference available on Amazon Web Services
The service will be available via AWS Bedrock and offer leading open-source LLMs and Amazon's Nova models at industry-leading inference speeds
AWS and Cerebras are collaborating on a new disaggregated architecture that pairs AWS Trainium with Cerebras WSE
This disaggregated approach delivers 5x more high-speed token capacity in the same hardware footprint compared to traditional architectures
The disaggregated architecture splits inference workloads: Trainium handles prefill (question processing) while Cerebras WSE handles decode (answer generation)
Cerebras CS-3 stores all model weights on-chip in SRAM, delivering thousands of times greater memory bandwidth than the fastest GPUs
The collaboration addresses the growing need for fast inference, particularly for AI agentic coding which generates 15x more tokens per query than conversational chat
AWS will support both aggregated and disaggregated configurations, allowing customers to route workloads based on their specific needs
The disaggregated mode connects Trainium and WSE via Amazon's high-speed EFA (Elastic Fabric Adapter) interconnect
Cerebras currently powers models from OpenAI, Cognition, and Meta at speeds up to 3,000 tokens per second
