## Key Points

llmfit is a terminal tool that helps users find the right LLM models for their specific hardware by detecting system RAM, CPU, and GPU capabilities
The tool scores models across quality, speed, fit, and context dimensions to recommend which models will actually run well on your machine
It ships with both an interactive TUI (default) and classic CLI mode, supporting multi-GPU setups, MoE architectures, and various quantization levels
llmfit supports multiple local runtime providers including Ollama, llama.cpp, and MLX, with integration for detecting installed models and downloading new ones
The tool features a unique Plan mode that inverts the analysis - instead of finding models that fit your hardware, it estimates what hardware is needed for specific model configurations
It includes a multi-dimensional scoring system that weights different aspects (quality, speed, fit, context) differently based on use case categories like General, Coding, Reasoning, Chat, Multimodal, and Embedding
Speed estimation is based on actual GPU memory bandwidth calculations with fallback constants for unrecognized hardware
The model database includes hundreds of models from HuggingFace, covering major providers like Meta, Mistral, Google, Microsoft, and many others
llmfit can also run as a REST API server for cluster scheduling and aggregation purposes
The tool is available via multiple installation methods (scoop, brew, curl) and supports Windows, macOS, and Linux with cross-platform hardware detection
