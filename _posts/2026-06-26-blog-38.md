## Key Points

- Fluree DB is a temporal, verifiable graph database with standards-compliant RDF support and git-like branching/merging capabilities
- Optimized for AI agents with integrated vector, text, and geo search, plus fine-grained access control without external dependencies
- Supports multiple query languages including RDF 1.1/1.2, SPARQL, JSON-LD, and Open Cypher
- Handles billions of graph facts on commodity hardware with benchmark performance 10.4x faster than competitors
- Includes time travel functionality allowing queries at any point in history via transaction numbers, timestamps, or commit IDs
- Features built-in full-text search (BM25) and vector similarity search (HNSW) integrated directly into the query engine
- Offers edge annotations for attaching metadata to relationships rather than just nodes
- Provides branch, rebase, merge, push, pull workflows similar to git but for data
- Implements fine-grained access control with triple-level policy enforcement
- Available as CLI, HTTP server, Rust library, and MCP server for AI assistant integration
