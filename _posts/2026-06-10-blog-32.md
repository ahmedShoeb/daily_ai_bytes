## Key Points

- Thousand Token Wood v2 transforms from a sandbox simulation into an interactive financial game where players act as 'Patron of the Wood' shadow financiers
- The system uses heterogeneous models from four different AI labs: OpenAI's gpt-oss-20b, OpenBMB's MiniCPM3-4B, NVIDIA's Nemotron-Mini-4B, and a custom fine-tuned Qwen 0.5B model
- Each woodland creature agent operates with a different AI model, creating genuine behavioral diversity rather than scripted interactions
- Key challenges included serving layer compatibility issues, particularly with vLLM requiring CUDA toolkit installation and different model-specific configuration requirements
- The game implements a strict information firewall system to prevent agents from accessing hidden truth flags about insider trading tips
- Memory management uses bounded sentiment summaries instead of full history to prevent prompt inflation in small models
- Real-world testing showed 100% reliability for the fine-tuned 0.5B model and zero information leaks of hidden tip flags
- The project demonstrates that heterogeneous agent systems can be manageable once serving infrastructure is standardized
