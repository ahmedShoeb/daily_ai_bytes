## Key Points

This is Claude Sonnet 5, built to be the most agentic Sonnet model yet with capabilities to make plans, use tools like browsers and terminals, and run autonomously at levels that previously required larger, more expensive models
Sonnet 5 significantly narrows the gap with Opus-class models, delivering performance close to Opus 4.8 but at lower prices
The model shows substantial improvements over Sonnet 4.6 in key areas like reasoning, tool use, coding, and knowledge work
Available across all plans from today (Free, Pro, Max, Team, Enterprise) with introductory pricing of $2 per million input tokens and $10 per million output tokens through August 31, 2026
Early access partners reported Sonnet 5 finishing complex tasks where previous Sonnet models would stop short and checking its own output without explicit prompting
Safety assessments found Sonnet 5 has lower rates of undesirable behaviors than Sonnet 4.6 and is generally safer in agentic contexts
The model has much lower ability to perform cybersecurity tasks compared to Opus models, with cyber safeguards enabled by default
Uses an updated tokenizer that improves performance but can increase token count by 1.0–1.35× depending on content type
Developers can access the model via the Claude API using claude-sonnet-5
Sonnet 5 represents a genuine step forward for enterprise teams managing complex workloads with strong performance where it counts
