## Key Points

Manus has launched Scheduled Tasks 2.0, a major upgrade to recurring work automation
The new version focuses on running automation with proper context rather than simply repeating tasks on a schedule
Scheduled work can now continue inside the same task context, maintaining connections to instructions, files, conversations, and previous results
Users can choose whether each scheduled run continues in the same task or starts as a separate task
Scheduled actions can now be added to web apps built with Manus for automatic data refreshes, script execution, and dashboard updates
Projects can have their setup reused across scheduled tasks, including files, skills, connectors, instructions, and output standards
New visibility features include side panels, schedule views, calendar views, and run cards to track upcoming and past executions
Skip confirmations option allows trusted workflows to proceed without manual approval for sending, publishing, or posting
Connectors enable scheduled tasks to use data from connected apps as relevant sources
The upgrade is now available to all Manus users across tasks, Projects, and web apps
