## Key Points

- claudetop is a monitoring tool that shows real-time Claude Code usage and costs in your terminal
- It addresses the problem of hidden costs in Claude Code where users can't see what they're spending in real time
- The tool provides session information including project location, model being used, duration, token counts, and real-time costs
- It includes cache efficiency metrics to show if tokens are being wasted or reused effectively
- Users can compare costs across different Claude models (Opus, Sonnet, Haiku) with cache-aware pricing
- The system provides smart alerts for budget overruns, low cache efficiency, high burn rates, and other issues
- Every session is automatically logged for historical analysis and cost tracking
- Includes iTerm2 integration for tab titles, status bar displays, and background color coding based on session state
- Supports customizable themes and plugin system for additional functionality like Spotify integration, CI status, weather, etc.
- Pricing updates automatically from the repository when Anthropic changes Claude pricing
