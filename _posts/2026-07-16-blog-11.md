## Key Points

-Learn how to switch to Claude Fable-5 model: use `/model` command in Claude Code and select Fable, or use model ID 'claude-fable-5' in API
-New dynamic workflows feature (research preview): Claude writes orchestration scripts to spin up coordinated subagents for complex tasks like bug hunts, migrations, or stress testing
-Try dynamic workflows by mentioning 'workflow' in prompts; they're powerful but can be expensive—start with scoped tasks to test
-New '/effort ultracode' level runs at high effort and lets Claude decide when a task needs a dynamic workflow
-Full-screen renderer improvements fix bugs like screen flickering—toggle with `/tui feedback` command
-Enhanced Claude responsiveness with streamed thinking & tool calls to prevent hanging appearance
-Security-guidance plugin for Claude Code helps identify and fix vulnerabilities while coding
-Security plugin runs on three levels: file edits, after model turns, and on commits—reduces security-related PR comments by 30-40%
-Starting June 15: paid Claude plans get dedicated monthly credits for programmatic usage (Agent SDK, claude -p, GitHub Actions, third-party apps)
-Programmatic usage now has separate budget from interactive subscription limits; usage pauses if credits run out unless usage credits are enabled
