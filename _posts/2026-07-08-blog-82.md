## Key Points

- Author provides comprehensive hardware guides for running state-of-the-art LLMs locally with budgets ranging from $2k to $40k
- $2k setup can run Qwen3.6-27B and local speech-to-text, while $40k setup approaches Claude Opus-level capabilities
- Custom 4x RTX PRO 6000 system with 384GB VRAM costs approximately $46k for GPUs plus $5.6k for base system components
- Uses PCIe4 switches from c-payne.com to enable direct GPU-to-GPU communication, reducing latency compared to routing through CPU
- System runs on 110V circuit with power-limited GPUs (350W each) to avoid installing 220V electrical infrastructure
- Achieves Gen4 line rate performance: 27.5 GB/s unidirectional, 50.4 GB/s bidirectional P2P bandwidth with sub-microsecond latency
- Includes detailed BIOS configurations, kernel parameters, and system optimizations for maximizing performance
- Provides ready-to-run Docker configurations for models like GLM-5.2-594B achieving ~80 tokens/second at 460k context length
- Also runs local speech-to-text using cohere-transcribe model requiring only ~11GB VRAM
- Complete system uses last-gen EPYC processor with DDR4 RAM to save costs while maximizing VRAM investment
