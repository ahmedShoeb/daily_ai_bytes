## Key Points

- Anthropic announced that Claude Code and Claude Cowork tools are being updated to accomplish tasks using your computer
- The AI can now open files, use browsers, and run developer tools
- The feature is currently in research preview for Claude Pro and Claude Max subscribers on macOS
- Claude will prioritize connectors to supported services like Google Workspace or Slack, but can execute tasks even without specific connectors
- The AI should ask for permission before taking actions, but Anthropic recommends avoiding sensitive information handling with this feature
- The feature supports Anthropic's Dispatch, allowing continuous conversations across phone and desktop
- Claude Cowork was introduced in January as a more casual-user-friendly version of Claude Code for programmers
