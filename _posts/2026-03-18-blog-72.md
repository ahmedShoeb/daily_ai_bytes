## Key Points

Google's Stitch AI design tool, rebranded from acquired Galileo AI, is undergoing a complete redesign revealed through TestingCatalog findings
The most striking change is moving from a flat canvas to a 3D workspace that creates a spatial design environment feeling
An AI agent appears as a compact, expandable widget that visually shifts position between design and prototype modes
Voice mode allows users to choose from multiple voice options for their agent, with the agent responding through spoken audio and glowing visual feedback
The AI agent can modify designs, explain choices, offer suggestions, and brainstorm as a conversational design partner
Prototyping features include screen-to-screen connections, QR code generation for user research, and 'imagine more screens' feature that suggests missing components
Biggest leap: Stitch can now generate fully functional React applications from selected screens, not just clickable prototypes
Working code can be exported to AI Studio or other tools, positioning Stitch as an end-to-end pipeline from concept to production
The overhaul integrates previously spotted features like studio mode, voice controls, and the Hatter agent into the tool's final form
The complete redesign could be showcased at Google I/O 2026 scheduled for May 19–20
