## Key Points

- Graphsignal is a production-scale inference profiling platform for optimizing AI performance across models, engines, GPUs, and accelerators
- Provides continuous, high-resolution profiling timelines showing operation durations and resource utilization across inference workloads
- Offers LLM generation tracing with per-step timing, token throughput, and latency breakdowns for major inference frameworks
- Includes system-level metrics for inference engines and hardware (CPU, GPU, accelerators)
- Features error monitoring for device-level failures and inference errors
- Provides inference telemetry for AI agents to identify bottlenecks and drive targeted improvements
- Supports CUDA 12.x and 13.x installations via uv tool or pip package manager
- Requires GRAPHSIGNAL_API_KEY environment variable for authentication with free account from graphsignal.com
- Has minimal performance impact on production systems, using low-overhead CUPTI APIs for CUDA kernel activity collection
- Only establishes outbound connections to api.graphsignal.com and does not record sensitive information like prompts and completions
