## Key Points

- Anthropic is building a tool called Bugcrawl within Claude Code that appears in the side navigation
- The tool allows users to select a repository and comes with a warning about high token consumption rates, suggesting starting with small repositories first
- Bugcrawl appears designed to hunt for general bugs and propose fixes across entire codebases, separate from the existing Security tab which focuses on vulnerabilities
- The move fits into Anthropic's recent Claude Code expansion which already includes Code Security and Code Review features
- The tool likely targets engineering teams on Team and Enterprise tiers who can better absorb the token costs
- No release window has been announced and the feature doesn't appear in production builds yet
- The article suggests Bugcrawl could potentially extend into end-to-end product testing where Claude could test user flows and report regressions
- This development aligns with the competitive landscape where companies like OpenAI, xAI, and Google are also building agents that work across full repositories
