## Key Points

RCLI is an on-device voice AI assistant for macOS that runs completely locally without cloud services or API keys
It provides a full STT (speech-to-text) + LLM + TTS (text-to-speech) pipeline running natively on Apple Silicon
Features sub-200ms end-to-end latency and supports 38 macOS actions via voice commands
Includes local RAG (retrieval-augmented generation) for querying personal documents like PDFs and DOCX files
Powered by MetalRT, a proprietary GPU inference engine built specifically for Apple Silicon that requires M3 or later chips
Installable via one command with curl or Homebrew, requiring macOS 13+ and Apple Silicon
Supports interactive TUI with push-to-talk, continuous voice mode, and one-shot text commands
Automatically falls back to llama.cpp engine on M1/M2 Macs while MetalRT requires M3 or later
Models take about 1GB download during initial setup with support for 20+ models across LLM, STT, and TTS
The project is open source under MIT license with MetalRT engine being proprietary software
