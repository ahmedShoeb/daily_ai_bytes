## Key Points

- Anthropic's Claude Mythos reportedly solved the Erdős unit-distance conjecture, an open mathematical problem in combinatorial geometry that dates back to 1946
- OpenAI had recently disproved this same problem, which was considered an AI math milestone
- Anthropic engineer Sholto Douglas described Mythos' solution as using a "cute, simple proof" and called it a sign of "serious overhang" in AI-driven math discoveries
- The team used a specialized test system where isolated Claude Code instances with Mythos access work on developing solution paths independently
- According to reports, Mythos frequently took different solution routes than OpenAI's model approach
- While mathematician Daniel Litt called the Mythos result "a bit worse" than OpenAI's, Mythos also reportedly found OpenAI's solution
- Anthropic published a proof version prepared by Opus 4.7
- Google DeepMind recently announced solving nine Erdős problems, but using the formal proof language Lean which some view as less impressive from an LLM purist perspective
