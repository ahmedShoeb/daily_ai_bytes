## Key Points

Pi Dynamic Workflows is a Pi extension that adds a workflow tool inspired by Anthropic's dynamic workflows in Claude Code
Instead of one assistant working sequentially, the model writes JavaScript scripts that fan out work across isolated subagents
The tool is ideal for codebase audits, multi-perspective reviews, large refactors, and fan-out research
Workflow scripts run in a Node vm sandbox with intentional restrictions to keep operations reproducible and secure
The system provides global functions like agent(), parallel(), pipeline(), and phase() for controlling workflow execution
Subagents run in fresh in-memory Pi sessions with standard coding tools and can return structured outputs via JSON Schema validation
Projects can include TypeScript hints for workflow globals using /// <reference types="pi-dynamic-workflows/workflow" />
The tool shows live progress with phases and can be canceled with Esc during execution
This is currently a prototype that doesn't yet support persisted/resumable runs or a /workflows manager
Installation is done via npm: pi install npm:pi-dynamic-workflows followed by /reload in Pi
