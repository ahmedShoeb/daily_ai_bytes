## Key Points

- RoadmapBench is a new benchmark for evaluating coding agents on long-horizon software development tasks
- The benchmark includes 115 tasks based on real open-source version upgrades across 17 repositories and 5 programming languages
- Each task requires implementing functionality from target versions with median modifications of 3,700 lines across 51 files
- The research highlights a gap in existing benchmarks that focus mostly on single-issue bug fixes from Python repositories
- Current frontier models struggle significantly, with Claude-Opus-4.7 resolving only 39.1% of tasks and the weakest achieving 5.2%
- The findings suggest long-horizon software development remains largely unsolved despite advances in coding agents
