## Key Points

)Cursor researchers have developed improved agent swarm technology that enables multiple AI agents to cooperate on complex tasks like building software from scratch
- The new system uses a two-tier approach: 'planner' agents powered by smart models break down tasks, while 'worker' agents powered by faster, cheaper models execute the work
- The swarm successfully built SQLite from scratch in Rust, passing 80% of the SQL test suite in 4 hours with Grok 4.5, compared to the old swarm which had to be paused
- Different model combinations showed similar quality results but dramatically different costs, ranging from $1,339 to $10,565 for the same task
- The system addresses coordination failures through mechanisms like shared design docs, third-party merge conflict resolvers, and automated file decomposition
- The agent swarm approach resembles a compiler that translates intent into executable work, with planners parsing goals into task trees
- Worker agents handled 69-90% of tokens but planners accounted for most costs due to higher per-token prices
- The new system dramatically reduced coordination problems, with only 47 conflicts on the most contested file vs 7,771 in the old system
- The swarm created a 'Field Guide' system where agents curate shared knowledge to help future agents, improving efficiency
- This approach raises the abstraction level in software engineering from individual lines of code to complete specifications
