## Key Points

Anthropic is releasing new connectors that allow Claude to integrate with creative software tools used by professionals
Key connector integrations include: Ableton, Adobe Creative Cloud (50+ tools), Affinity by Canva, Autodesk Fusion, Blender, Resolume Arena/Wire, SketchUp, and Splice
Claude can help creative professionals learn and master complex software, write scripts/plugins, bridge tools in pipelines, and handle repetitive production tasks
Blender connector uses MCP protocol and gives Claude access to Blender's Python API for 3D modeling and scene analysis
Anthropic made a donation to support Blender's Python API development and the connector is accessible to other LLMs due to Blender's open-source commitment
The company is working with art/design programs at RISD, Ringling College, and Goldsmiths University to support creative computation curricula
Claude Design (new product from Anthropic Labs) helps explore software experience ideas and exports results to tools like Canva
