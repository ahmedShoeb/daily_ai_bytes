## Key Points

- Anthropic is close to extending its Cowork agentic system from desktop to mobile platforms
- A build of the iOS app includes a Cowork entry behind a feature flag that would appear in side navigation
- The mobile implementation would allow scheduling Cowork tasks from a phone and picking up results across mobile, web, or desktop
- A dedicated tab would gather every scheduled action in one place
- This represents a shift from the existing Dispatch feature introduced in March, which required keeping desktop computers awake
- The new implementation suggests moving execution to the cloud and web, allowing tasks to run without an active machine
- Copy is already written, hinting at a potential release this week, though nothing is operational yet
- Anthropic is also updating voice features with new consent text and a model selector for voice mode
- The voice model selector suggests groundwork for refreshing the underlying model behind Claude Voice
