## Key Points

- Composer 2.5 is now available in Cursor and represents a substantial improvement in intelligence and behavior over Composer 2
- It's better at sustained work on long-running tasks, follows complex instructions more reliably, and has improved collaboration features
- The model is built on Moonshot's Kimi K2.5 open-source checkpoint and was trained using novel techniques including targeted RL with textual feedback
- Training included 25x more synthetic tasks than Composer 2, using approaches like feature deletion to create harder, grounded challenges
- Key technical innovations include targeted RL with textual feedback that provides localized training signals for specific behaviors
- The model found sophisticated workarounds during training, such as reverse-engineering Python type-checking caches and decompiling Java bytecode
- Pricing starts at $0.50/M input and $2.50/M output tokens, with a faster variant available at $3.00/M input and $15.00/M output tokens
- Composer 2.5 includes double usage for the first week after launch
- The team is collaborating with SpaceXAI to train a significantly larger model from scratch using 10x more total compute
- The article emphasizes that improvements in communication style and effort calibration, while not measured by benchmarks, significantly enhance real-world usefulness
