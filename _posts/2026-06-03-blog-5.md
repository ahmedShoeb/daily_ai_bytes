## Key Points

Google is working on three major new features for NotebookLM: Personal Preferences, Connectors, and Canvas
Personal Preferences will allow NotebookLM to learn from user activity and build editable personas to adjust tone and technical depth based on how users work
Connectors will pull outside data from services like Calendar, Gmail, and Drive into notebooks, similar to MCP (Model Context Protocol)
Canvas is a headline feature that turns sources into custom interactive artifacts like timelines, explainer web pages, light games, or visualizers guided by prompts
These features extend NotebookLM's existing outputs including infographics, slide decks, data tables, and mind maps
NotebookLM now lives inside Gemini, allowing users to work across sources without copying material between tools
The combination aims to turn a source-grounded reader into a workspace for building structured, visual experiences on documents
NotebookLM has already moved to Gemini 3 late last year, with Gemini 3.5 Flash likely being the next base model
The features have been surfacing in recent builds but lack a firm release timeline
The features represent Google's push to deepen NotebookLM's integration with user workflows and external data sources
