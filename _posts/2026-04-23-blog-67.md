## Key Points

- This is Module 4 of the 'Building Agentic Systems on Amazon Web Services (AWS)' workshop series focused on multi-agent architectures
- The workshop provides a practical deep dive into multi-agent coordination, task sharing, failure recovery, and scaling beyond prototypes using AWS production infrastructure
- Participants will learn to model agent workflows as directed state graphs using LangGraph, which has over 38 million monthly downloads
- The workshop covers adding durable orchestration with AWS Step Functions for features like idempotency, automatic retries, and visual workflow debugging
- Attendees will learn event-driven messaging using Amazon EventBridge and Amazon MQ for decoupled inter-agent communication with reliable delivery
- The curriculum includes production architectural patterns like shared-state isolation, task decomposition strategies, and graceful degradation to prevent cascading failures
- The workshop features senior technical speakers from Amazon including Dr. James Bland, WW Tech Lead for Data & AI, and Mike Brugnoni, Sr. Solutions Architect
- The session includes both practical hands-on experience and a companion guide for high-level overview learning
