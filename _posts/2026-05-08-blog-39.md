## Key Points

- Google has expanded the Gemini API's File Search tool to support multimodal data processing
- The tool now natively processes both text and visual data together using Gemini Embedding 2 model
- Custom metadata functionality allows attaching key-value labels to unstructured data for better organization
- Page citations have been introduced to improve grounding and transparency by linking responses to source pages
- Developers can build retrieval-augmented generation (RAG) systems with visual data, enabling search by emotional tone or visual style
- Metadata filters help reduce noise by scoping queries to relevant data slices, improving speed and accuracy
- The update benefits both prototyping weekend projects and scaling production applications for thousands of users
- File Search handles infrastructure heavy lifting so developers can focus on building products
- The tool now provides contextual awareness for agents when searching through image archives
