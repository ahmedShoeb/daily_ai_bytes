## Key Points

AutoTTS is a novel framework that automatically discovers efficient test-time scaling (TTS) strategies for LLMs through environment-driven automatic search
The system turns TTS strategy design from hand-crafted heuristics to automatic discovery - humans only construct an offline replay environment, and a coding agent iteratively proposes and refines code-defined controllers
The discovered 'Confidence Momentum Controller' (CMC) saves approximately 69.5% of tokens compared to SC@64 while matching mean accuracy across four backbone model scales
Key innovations include trend-based stopping using exponential moving averages of confidence, coupled width-depth control, alignment-aware depth allocation, and conservative branch abandonment
The framework achieves optimization through iterative program search rather than gradient updates or backpropagation - using 0 LLM calls during discovery evaluation (only replays cached segments)
One full discovery run costs an estimated $39.9 and takes 160 minutes wall-clock time
AutoTTS was optimized on AIME24 replay constructions and shows strong performance on held-out benchmarks including AIME25 and HMMT25
The system treats adaptive test-time inference as allocating finite budget over branches in fixed-length intervals with a Markov Decision Process formulation
The Beta parameterization allows controllers to export a single scalar that deterministically schedules all internal hyperparameters for smooth scaling behavior
The repository includes complete implementation with evaluation frameworks, replay environments, and tools for both evaluating pre-discovered controllers and running new discovery workflows
