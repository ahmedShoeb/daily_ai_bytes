## Key Points

Unsloth Studio is a new open-source, no-code web UI for training, running, and exporting AI models locally
Allows running GGUF and safetensor models on Mac, Windows, and Linux with 2x faster training and 70% less VRAM
Supports text, vision, TTS audio, and embedding models without accuracy loss
Features include local model execution, self-healing tool calling/web search, and code execution (Bash + Python)
No dataset needed - auto-creates datasets from PDF, CSV, JSON, DOCX, TXT files
Includes Data Recipes that transform documents into usable datasets using graph-node workflow
Provides model arena for comparing different models side-by-side
Works 100% offline with token-based authentication and privacy-first approach
Recent updates include fixes for Windows, MacOS CPU support, and Intel GPU training compatibility
Uses dual-licensing model (Apache 2.0 + AGPL-3.0) to support ongoing development while remaining open source
