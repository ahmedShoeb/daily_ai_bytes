## Key Points

- Agent-World is a self-evolving training arena that unifies scalable environment synthesis with continuous agent training for advancing general AI agent intelligence
- The system autonomously mines real-world tool ecosystems (MCP servers, tool documentation, industrial PRDs) to build over 2,000 realistic environments across 20 categories with 19K+ validated tools
- Agent-World uses two complementary task synthesis methods: graph-based task synthesis and programmatic task synthesis with executable Python solutions for verifiable evaluation
- The system features a continuous self-evolving agent training loop that diagnoses agent weaknesses through dynamic evaluation and generates targeted tasks for iterative improvement
- Across 23 challenging agent benchmarks, Agent-World consistently outperforms strong proprietary models and environment scaling baselines, achieving up to 65.4% accuracy on τ²-Bench with the 14B model
- Key findings show that even advanced proprietary models like GPT-5.2 High struggle with complex agentic tool-use scenarios, achieving only 53.1% on MCP-Mark benchmarks
- The research reveals clear scaling trends where performance improves as training environments increase from 0 to 2000, with scores rising from 18.4% to 38.5% across four domains
- Self-evolution rounds provide monotonic gains, with the largest improvements on MCP-Mark which requires strong state tracking and interaction with realistic MCP server environments
- Agent-World demonstrates that scalable realistic environments are critical infrastructure for advancing general agent capabilities beyond mere data sources
- The system represents a significant advancement in creating practical, continuously learning AI agents that can interact with real-world tools and services
