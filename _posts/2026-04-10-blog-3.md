## Key Points

- Open-source tool for fine-tuning Google's Gemma models (Gemma 3n and Gemma 4) on text, images, and audio data
- Designed specifically for Apple Silicon Macs (MPS-native), eliminating the need for NVIDIA GPUs or cloud rentals
- Supports LoRA fine-tuning across three modalities: text-only instruction/completion, image captioning/VQA, and audio transcription
- Includes streaming from cloud sources like GCS and BigQuery, allowing training on terabytes of data without local storage constraints
- Features real-time browser-based visualization during training showing loss curves, attention heatmaps, gradient strength, memory usage, and token predictions
- Uses CSV-based dataset structure with simple configuration via profiles in config/config.ini
- Supports domain-specific adaptation for medical, legal, call-center audio, visual domains like receipts/charts, and underrepresented languages
- Entirely private, on-device pipeline - data never leaves the machine and weights never touch third-party APIs
- Requires Python 3.10+, macOS 12.3+, and native arm64 Python installation (not Rosetta)
- Includes wizard interface for guided setup and comprehensive CLI commands for preparation, training, evaluation, and export
