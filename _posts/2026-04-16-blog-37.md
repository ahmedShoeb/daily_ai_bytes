## Key Points

- Microsoft is testing OpenClaw-like features for integration into its Microsoft 365 Copilot tool, aimed at enterprise customers with better security controls than the open source OpenClaw.
- The new agent would essentially be a version of 365 Copilot that is always working and able to complete multistep tasks over long periods.
- This effort joins other agentic tools Microsoft has recently announced, including Copilot Cowork (which takes actions in Microsoft 365 apps) and Copilot Tasks (for prosumer-oriented tasks like organizing email and travel).
- Microsoft has partnered with Anthropic and added Claude as an option for Copilot Cowork, though unlike OpenClaw, current Microsoft agents run in the cloud rather than locally.
- The company is expected to showcase this new Claw (or an upgraded version) at Microsoft Build conference in June.
- While OpenClaw runs locally and is popular on Mac Mini computers, Microsoft hasn't confirmed whether its version will run locally or just adopt certain OpenClaw features.
- Microsoft confirmed it's 'experimenting' with broader orchestration and autonomy in AI experiences while focusing on security, governance, and trust.
