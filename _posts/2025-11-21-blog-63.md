## Key Points

Anthropic's Claude Sonnet 4.5, Haiku 4.5, and Opus 4.1 models are now in public preview in Microsoft Foundry.
This allows Azure customers to build production applications and enterprise agents using Claude within their existing Microsoft ecosystem.
Claude models can also be used with Claude Code, an AI coding agent, in Microsoft Foundry.
Claude powers the Researcher agent for complex research and enables custom agent development in Copilot Studio for Microsoft 365 Copilot.
Microsoft's Agent Mode in Excel now includes an option to use Claude to generate formulas, analyze data, identify errors, and iterate on solutions.
The integration removes procurement barriers for enterprises already invested in Microsoft Foundry and Microsoft 365 Copilot.
Claude is available in Microsoft Foundry via serverless deployment and is eligible for Microsoft Azure Consumption Commitment (MACC).
Developers can access Claude using Python, TypeScript, and C# SDKs with Microsoft Entra authentication.
Sonnet 4.5 is ideal for coding and complex agents, Haiku 4.5 for speed and cost-efficiency in high-volume applications, and Opus 4.1 for specialized reasoning tasks.
All models support various Claude Developer Platform capabilities like code execution, web search, citations, and vision.
