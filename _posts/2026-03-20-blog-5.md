## Key Points

OpenAI has released GPT-5.4 mini and nano, their most capable small models designed for high-volume workloads
GPT-5.4 mini offers significant improvements over GPT-5 mini in coding, reasoning, multimodal understanding, and tool use while running more than 2x faster
GPT-5.4 mini approaches the performance of the larger GPT-5.4 model on several evaluations including SWE-Bench Pro and OSWorld-Verified
GPT-5.4 nano is the smallest and cheapest version of GPT-5.4 for tasks where speed and cost matter most, ideal for classification, data extraction, ranking, and simpler coding subagents
The models are optimized for latency-sensitive applications like coding assistants, subagents, computer-using systems, and real-time multimodal applications
GPT-5.4 mini is available today in the API, Codex, and ChatGPT, with API pricing at $0.75 per 1M input tokens and $4.50 per 1M output tokens
In Codex, GPT-5.4 mini uses only 30% of the GPT-5.4 quota, allowing developers to handle simpler coding tasks at about one-third the cost
GPT-5.4 nano is only available in the API and costs $0.20 per 1M input tokens and $1.25 per 1M output tokens
Both models support text and image inputs, tool use, function calling, web search, file search, computer use, and skills with a 400k context window
The models are particularly effective in coding workflows that benefit from fast iteration, handling targeted edits, codebase navigation, front-end generation, and debugging loops with low latency
