## Key Points

- Sapiens2 is a family of high-resolution transformer models pretrained on 1 billion human images
- Developed by Facebook Research, it achieves state-of-the-art performance across diverse human-centric tasks
- Supported tasks include pose estimation, body-part segmentation, surface normals estimation, and pointmap estimation
- The models range from 0.1B to 5B parameters, with different sizes offering different computational trade-offs
- All models use patch size 16 and are trained at 1024×768 resolution except Sapiens2-1B (4K) trained at 4096×3072
- The project provides standalone backbone files that can be integrated into other projects without the full repository
- Requires Python ≥3.12 and PyTorch ≥2.7 for installation
- The code is licensed under the Sapiens2 License and available on GitHub with comprehensive documentation
- Built on contributions from DINOv3, OpenMMLab, and Accelerate frameworks
- Includes pre-trained checkpoints for different model sizes and tasks available for download
