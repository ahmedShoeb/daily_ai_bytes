## Key Points

- Proposes a test-time scaling framework for agentic coding based on compact representations of rollout trajectories
- Converts each rollout into structured summaries preserving salient hypotheses, progress, and failure modes while discarding low-signal details
- Enables two inference-time scaling methods: Recursive Tournament Voting (RTV) for parallel scaling and Parallel-Distill-Refine (PDR) for sequential scaling
- Improves performance of frontier coding agents on SWE-Bench Verified and Terminal-Bench v2.0 benchmarks
- Claude-4.5-Opus improved from 70.9% to 77.6% on SWE-Bench Verified using the method
- Terminal-Bench v2.0 performance improved from 46.9% to 59.1%
- Identifies that test-time scaling for long-horizon agents is fundamentally about representation, selection, and reuse
- Addresses challenge that existing methods work best for short, bounded outputs, not long trajectories of agent actions
