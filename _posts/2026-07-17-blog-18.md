## Key Points

The focus has shifted from individual agents to the systems that surround them, with emphasis on harness engineering for self-improvement and reliable agent management
Loop engineering has emerged as the new control layer, distinguishing between inner agent loops and outer human oversight loops for better system management
AI engineering is entering enterprises through forward deployed engineers who implement software factories and manage agentic ecosystems at scale
Coding agents are replacing traditional IDEs as the primary developer interface, handling broader objectives across multiple files with autonomous debugging and iteration
Agent platforms are increasingly built around skills - reusable, declarative workflows that encode engineering best practices and reduce orchestration code requirements
