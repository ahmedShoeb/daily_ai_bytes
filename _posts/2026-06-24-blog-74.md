## Key Points

- ENPIRE is a framework that enables coding agents to autonomously improve robotic manipulation policies in the real world through a repeatable feedback loop
- The system has four core modules: Environment (EN), Policy Improvement (PI), Rollout (R), and Evolution (E) for automated policy refinement
- ENPIRE enables frontier coding agents to achieve 99% success rates on challenging dexterous manipulation tasks like PushT, pin insertion, and zip tie cutting
- The system works by automatically resetting scenes, executing policies, verifying outcomes, and allowing coding agents to analyze logs and refine algorithms
- Researchers tested multiple coding agents including Codex with GPT-5.5, Claude Code with Opus 4.7, and Kimi Code with Kimi K2.6
- ENPIRE can scale across robot fleets, with researchers introducing Mean Robot Utilization (MRU) and Mean Token Utilization (MTU) metrics to measure efficiency
- The system transforms real-world robot learning into a controllable optimization procedure that minimizes human effort while allowing systematic experimentation
- While scaling robot fleets accelerates research, it also drives higher token consumption as more agents coordinate and summarize peer branches
- The research also includes simulation results in RoboCasa to separate agent-driven research behavior from hardware throughput limitations
- This work suggests a practical and scalable path toward autonomously advancing robotics capabilities in physical environments
