## Key Points

- NotebookLM mobile app has started rolling out with an MVP feature set
- Users can listen to Audio Overviews offline by downloading them
- Audio Overviews can be played in the background while using other apps
- Users can add sources from anywhere on their device via share icon
- Current supported inputs: websites, PDFs, and YouTube videos
- More input types will be added soon
- The app is available on mobile devices
- The initial release is focused on core functionality with more features coming
