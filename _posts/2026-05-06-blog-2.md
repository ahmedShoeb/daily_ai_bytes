## Key Points

OpenAI released a major update to its Codex desktop app featuring animated pixel-art pets that live as screen overlays even when minimized
Pets display status messages and serve as a two-way communication channel - clicking on them opens replies back to the AI agent
A new Hatch skill converts any uploaded image into a custom animated pet, which can be saved and shared
Community-driven directories like PetShare and PetDex emerged to share custom pet creations
Codex now auto-detects and imports configuration files from other coding agents including Claude Code's CLAUDE.md
This cross-agent config portability makes switching between AI agents mid-project less frustrating
New dictation dictionary feature allows users to pre-load custom abbreviations and phrases for better voice input accuracy
The update shows OpenAI positioning Codex from a coding tool into a comprehensive desktop super-app
The strategy focuses on charm, stickiness, and being an OS layer for ChatGPT alongside Atlas
