## Key Points

- TriAttention is an open-source research project that achieves 2.5x throughput boost and 10.7x KV cache compression on long reasoning tasks with no accuracy loss
- The method uses trigonometric frequency-domain compression to reduce KV memory requirements while maintaining accuracy comparable to Full Attention
- Developed by researchers from MIT, NVIDIA, and Zhejiang University with equal contributions
- Compatible with vLLM server and exposes OpenAI-compatible API for integration with tools like OpenClaw
- Supports multiple models including Qwen3-8B, DeepSeek-R1 variants, and provides precomputed frequency statistics for supported models
- Recent update includes experimental MLX and TurboQuant support for Apple Silicon (M1/M2/M3/M4)
- In benchmark tests, TriAttention maintains 40.8 accuracy on AIME25 (same as Full Attention) while achieving significant performance improvements
- Includes environment variables for fine-tuning compression parameters like KV budget, window size, and pruning modes
- Provides guidance for different workloads: interactive chat requires larger KV budgets (12k) vs offline benchmarks
- Licensed under Apache License 2.0 and has been published as arXiv preprint (2604.04921)
