## Key Points

- Source code for Claude Codes, Anthropic's AI coding assistant, appears to have been leaked online
- The leaked material includes core parts of Claude Codes such as the web client, backend services, and agent models
- Some files appear to date back to November 2024, suggesting the leak might be recent but not from the latest version
- The leak surfaced on the same day Anthropic launched its new Claude 3.7 Sonnet model, potentially overshadowing the announcement
- The source code was posted on popular code-sharing platforms but has since been taken down
- Anthropic has not yet publicly commented on the alleged leak
- This incident highlights growing concerns about AI model security as these systems become more powerful and widely deployed
- Experts note that source code leaks could help competitors understand proprietary technologies and accelerate their own development
- The timing and content suggest this may be a calculated leak rather than a random breach
- The leak raises questions about internal security practices at AI companies and the value of protecting AI intellectual property
