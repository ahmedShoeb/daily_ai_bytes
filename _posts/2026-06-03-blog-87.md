## Key Points

- Microsoft plans to launch a unified Copilot super app, potentially announced at Build 2026 conference on June 2 in San Francisco
- The app includes a Scout always-on agent within an Autopilot section and two new tabs: a coding surface and a Cowork tab
- The coding surface features GitHub Copilot integration with work tree selection, local/remote repository access, model selector, and scheduled-tasking Routines
- The Cowork tab aggregates data from multiple sources and suggests prompts for tasks like calendar preparation or company research
- The app appears to run in Edge via URL, raising questions about whether it will access local desktop files or remain remote
- Microsoft is reorganizing scattered Copilot tools into a single unified app to boost adoption, following similar patterns by OpenAI and Anthropic
- The app's shell has been internally referred to with the slogan "Delivering one Copilot"
- Teams integration suggests Scout could run remotely, potentially offering chat-app controls similar to popular AI agents
- The app is targeted for late summer release, with a likely preview at Build conference
