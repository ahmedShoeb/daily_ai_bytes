## Key Points

- CUGA (Configurable Generalist Agent) is IBM's open-source agent harness that handles planning, execution loops, tool calls, and state management, allowing developers to focus on tool lists and prompts
- The framework demonstrates productivity through cuga-apps: two dozen small, working apps each contained in a single FastAPI file that wrap a CugaAgent
- CUGA includes long-horizon planning with variable management and self-correction, topping agent benchmarks like AppWorld and WebArena
- The framework offers interchangeable tools (OpenAPI, MCP, and LangChain functions), declarative guardrails, multi-agent delegation, and one-environment-variable provider switching
- CUGA provides a governance system built into the runtime with six policy types that can be attached to agents for production safety
- The framework enables multi-agent systems through delegation to specialists, keeping planning surfaces manageable regardless of tool count
- CUGA supports on-the-job learning with ALTK-Evolve framework where agents refine skills from their own runs
- IBM Sovereign Core builds on CUGA to run agents under Boundary Isolation with data, control plane, and execution engine inside logical boundaries
- The live gallery showcases practical applications including IBM Cloud advisor, movie recommender, web researcher, and a seven-agent lead-gen system
- The framework is model-agnostic, supporting OpenAI, Anthropic, watsonx, Ollama, and IBM's gpt-oss-120b through simple configuration
