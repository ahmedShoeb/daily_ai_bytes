## Key Points

This is a hands-on technical workshop focused on multi-agent architectures as part of the Building Agentic Systems on AWS series.
The workshop features LangGraph for multi-agent orchestration with 38M+ monthly downloads alongside AWS services like Step Functions, EventBridge, and Amazon MQ.
Participants will learn to model agent workflows as directed state graphs with LangGraph, defining state schemas and conditional routing across specialized agents.
The workshop teaches how to add durable orchestration with AWS Step Functions for built-in idempotency, automatic retries, and state persistence across agent failures.
Content covers wiring event-driven messaging with Amazon EventBridge and Amazon MQ for reliable delivery using AMQP/MQTT protocols.
Production architectural patterns are covered including shared-state isolation, task decomposition strategies, and graceful degradation to prevent cascading failures.
The webinar aims to teach how multiple agents coordinate tasks, share state, recover from failures, and scale beyond prototypes using AWS production infrastructure.
Speakers include Dr. James Bland (WW Tech Lead, Data & AI, Amazon) and Mike Brugnoni (Sr. Solutions Architect, Amazon).
