## Key Points

- Xiaomi and inference partner TileRT achieved over 1,000 tokens per second on a 1-trillion-parameter model, a first at that scale using standard 8-GPU commodity hardware
- The speed dramatically outperforms current models: GPT-5.5 (68 tokens/sec), Claude Opus (71 tokens/sec), Gemini Flash (192 tokens/sec) compared to MiMo's 1,000+ tokens/sec
- The breakthrough comes from two key techniques: FP4 quantization of expert layers and DFlash speculative decoding that proposes full token blocks instead of one at a time
- The model matches Claude Opus on coding benchmarks while costing significantly less: MiMo at $0.87/million tokens output vs Opus at $25/million tokens
- Xiaomi achieved this through 'extreme model-system codesign' using software tricks alone on commodity GPUs, not requiring custom chips
- The UltraSpeed mode accelerates the exact MiMo V2.5 Pro model, not a stripped-down version, with near-zero quality loss from the optimizations
- A limited API trial runs June 9-23 at 3x standard MiMo rates but offers roughly 10x the generation speed
- This speed breakthrough enables new applications like parallel reasoning paths, fraud detection, trading signals, and real-time agent loops that were previously latency-constrained
