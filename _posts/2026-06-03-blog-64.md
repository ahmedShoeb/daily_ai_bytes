## Key Points

- OpenAI Robotics is actively hiring full-stack hardware, operations, systems, and ML engineers to build robots useful for society
- The robotics initiative evolved from OpenAI's world simulation research program led by Aditya Ramesh
- Short-term focus: robots to support skilled workers building infrastructure
- Long-term vision: personal robots for everyone to handle any needed tasks
- OpenAI announced raising $110 billion in funding from Amazon, NVIDIA, and SoftBank
- Partnership with Amazon for enterprise products and stateful runtime environments
- Continued relationship with Microsoft with stateless API exclusive to Azure
- Introducing GPT-5 as an integrated model that determines when to think harder, available to free tier users with reasoning capabilities
- New model called o1, described as OpenAI's most capable and aligned model yet, capable of general-purpose complex reasoning
- o1-preview and o1-mini available for ChatGPT Plus/Team users and API tier 5 users
