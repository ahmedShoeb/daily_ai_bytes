## Key Points

LaunchDarkly introduces AgentControl, a platform for managing AI agents in production environments
The platform addresses the problem of agent behavior shifting unexpectedly in production by automatically fixing issues in real time
AgentControl escalates to more capable configurations within the same conversation turn when responses drop below quality thresholds, preventing customers from seeing bad outputs
The system allows developers to replace hardcoded configurations with a single SDK call and manage prompts, models, and tools centrally instead of scattering them across services
Key features include: centralized configuration management, offline evaluations before deployment, progressive rollouts with automatic rollback on failure, comprehensive tracing and monitoring, and A/B/multi-armed bandit experiments
The platform is built on LaunchDarkly's infrastructure that serves 50 trillion flag evaluations daily and offers enterprise-grade reliability, security, and compliance certifications
