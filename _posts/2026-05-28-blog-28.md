## Key Points

Models.dev is a comprehensive open-source database of AI model specifications, pricing, and capabilities
The project addresses the lack of a single unified database for information about all available AI models
The database is community-contributed and also used internally by opencode
Data is accessible via a public API and stored as TOML files organized by provider and model
The platform allows users to look up models using the Model ID field that matches AI SDK identifiers
Provider logos are available as SVG files through a separate endpoint
The project provides detailed contribution guidelines for adding new providers and models
Contributors can add models by creating TOML files with specifications like cost, limits, modalities, and capabilities
There's an automated GitHub Action that validates submissions against a defined schema
The database supports model inheritance through an 'extends' feature for wrapper providers that mirror other models
