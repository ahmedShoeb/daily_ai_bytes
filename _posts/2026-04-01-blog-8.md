## Key Points

- Claude Code offers three scheduling options: Cloud, Desktop, and /loop, each with different capabilities and requirements
- Cloud scheduled tasks run on Anthropic's infrastructure without requiring your machine to be on or an open session
- Scheduled tasks can automate routine development work like PR reviews, CI failure analysis, documentation syncing, and dependency audits
- Tasks are created through web interface, desktop app, or CLI with conversational setup via the /schedule command
- Each task requires a self-contained prompt that specifies exactly what Claude should do during each run
- Tasks can work with GitHub repositories, create branches, and push changes (defaulting to claude/-prefixed branches)
- Environment configuration includes network access, environment variables, and setup scripts for each task
- Connectors allow tasks to interact with external services like Slack, Linear, or Google Drive
- Minimum scheduling interval is 1 hour for cloud tasks, but desktop and /loop tasks can run as frequently as every minute
- Past runs are accessible as regular sessions where you can review Claude's work and create pull requests
