## Key Points

- Proposes a novel framework called General Agentic Memory (GAM) to address limitations of static memory systems in AI agents
- GAM follows a 'just-in-time compilation' principle, creating optimized contexts at runtime while keeping only simple but useful memory offline
- Uses a duo-design with two key components: Memorizer and Researcher
- Memorizer highlights key historical information using lightweight memory while maintaining complete history in a universal page-store
- Researcher retrieves and integrates useful information from the page-store guided by pre-constructed memory for online requests
- Framework leverages agentic capabilities and test-time scalability of frontier large language models (LLMs)
- Enables end-to-end performance optimization through reinforcement learning
- Experimental study shows GAM achieves substantial improvements on various memory-grounded task completion scenarios compared to existing memory systems
- Submitted to arXiv on November 23, 2025, in the Computer Science > Computation and Language category
