## Key Points

- Researchers introduce Simple Self-Distillation (SSD), a method that improves large language model (LLM) code generation using only the model's own outputs
- SSD works by sampling solutions from the model with specific temperature and truncation settings, then fine-tuning on those samples using standard supervised fine-tuning
- The method improved Qwen3-30B-Instruct from 42.4% to 55.3% pass@1 on LiveCodeBench v6 test suite
- Gains were particularly concentrated on harder programming problems
- SSD generalizes across different model architectures (Qwen and Llama) and model sizes (4B, 8B, and 30B)
- The technique works with both instruct-tuned models and thinking variants
- Researchers found SSD works by addressing a precision-exploration conflict in LLM decoding
- The method reshapes token distributions contextually, suppressing distracting options when precision is needed while preserving useful diversity when exploration is beneficial
- SSD offers a complementary post-training approach for improving LLM code generation without requiring verifiers, teacher models, or reinforcement learning
