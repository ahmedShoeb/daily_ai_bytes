## Key Points

TinyFish launched Bigset, an open-source multi-agent system that converts plain language sentences into structured datasets pulled from the live web
Bigset automatically infers schema, sends autonomous agents to research on real web pages, verifies findings against sources, and deduplicates data
The system uses two agent roles: an orchestrator agent for discovery and sub-agents that research individual entities with tight 6-tool-call budgets
Users can set refresh cadences from 30 minutes to weekly to keep datasets current without manual intervention
Bigset is licensed under AGPL-3.0, runs self-hosted via Docker, and uses Claude Sonnet 4.6 for schema inference and Qwen3.7-max for agents
The free tier provides 2,500 row operations per month, with datasets taking 2-5 minutes to build and working best on topics with public web data
TinyFish is backed by $47 million in Series A funding led by ICONIQ, with enterprise clients including Google, DoorDash, and Amazon
The platform has processed over 40 million agent operations and Bigset is built on TinyFish's enterprise web infrastructure
Bigset ships with 9 curated public datasets covering AI companies hiring, GPU prices, model pricing, and top open-source repositories
The project represents an open-source alternative to proprietary natural-language dataset tools with no per-seat pricing or domain restrictions
