## Key Points

-First attempt at AI agent research using /deep-research tool burned through Claude Max 5x token limit in just 30 minutes with no usable results
-Created a multi-vendor pipeline using existing subscriptions (Claude, Codex, Antigravity) to leverage shared memory and reduce costs
-Implemented a model-orchestration pattern assigning different models to specific roles based on strengths and costs
-Developed a Bash script wrapper to run Codex and Antigravity as headless subagents orchestrated by Claude
-Added verification rules to reduce hallucinations: separate models for finding vs verifying, mandatory URLs/primary source quotes
-Moved /deep-research to the end of the pipeline to work on verified findings instead of open-ended exploration
-Discovered surprising insights about token economics including harness impact, context compaction pitfalls, and billing discrepancies
-Maintains a hybrid approach where agents do heavy lifting but humans verify results and improve the pipeline
-Achieved 10x longer research runs without additional costs by intelligently distributing work across subscriptions
