## Key Points

- LangSmith partnered with Fireworks to build an efficient Trace Judge that detects "Perceived Error" in agent traces
- They fine-tuned a Qwen-3.5-35B model to match or exceed frontier model performance while being up to 100x cheaper
- "Perceived Error" measures when users think the AI made a mistake or needed correction, which is different from objective correctness
- The model was trained on multi-turn traces from LangChain's Docs Q&A agent and no-code tool Fleet datasets
- Interestingly, the model trained on one dataset transferred well to the other, showing good generalization capabilities
- The fine-tuned Qwen model outperformed frontier models like Claude Haiku, Sonnet, and Opus on the evaluation tasks
- This approach demonstrates that specialized fine-tuned models can be both high-performing and cost-effective for large-scale trace analysis
- The model uses only Human and AI messages (excluding tool calls) to detect user corrections, rejections, and repeated requests as error signals
- LangChain plans to roll out this perceived error model to customers in the coming weeks and months
